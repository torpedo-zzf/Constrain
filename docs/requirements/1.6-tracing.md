# 1.6 全链路可观测（Tracing）

## 核心职责
跨组件、跨服务的Trace ID生成与传递，与OpenTelemetry兼容。

## 详细需求

| 需求项 | 规格 |
|--------|------|
| Trace ID | W3C Trace Context标准，32位hex，支持Span ID和Parent Span ID |
| 上下文传递 | 通过Event消息体trace_id字段携带 |
| OTel兼容 | 提供Exporter导出到Collector/Jaeger/Zipkin |
| 日志关联 | 所有日志自动注入trace_id |
| 轻量模式 | 单机模式不依赖外部Collector |
| 采样策略 | 全量采样、概率采样、错误强制采样 |

## 使用示例

from agentic_framework.tracing import Tracer

trace_context = Tracer.extract_or_create(event)
with Tracer.start_span(trace_context, "skill_execution"):
    result = await skill.execute(input_data, parameters, trace_context.trace_id)
